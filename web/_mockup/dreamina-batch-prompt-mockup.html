  • 高密度 — 8 条都不需要滚动, 一眼看全
  • 对齐感强 — 序号一列, 像在填表, 不容易遗漏
  • 轻量 — UI 占用最少, 跟单跑模式视觉接近

适合: prompt 数量多 (10+), 每条短小, 一次跑完不需要单独调。
"明显区分" 程度中等 — 行有分割但视觉密度高。
